#### 03 / Como o trabalho acontece

**Problema primeiro** — tecnologia é meio. Antes de qualquer linha de código, o processo, o gargalo e o resultado esperado săo mapeados.

**Simples e sustentável** — a melhor soluçăo é a que a equipe consegue operar amanhă. Sem complexidade desnecessária.

**Dados confiáveis** — dashboard bonito com número errado é pior que planilha. Qualidade e rastreabilidade vęm antes da estética.
